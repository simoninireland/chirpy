chirpy: The St Andrews bird counting experiments
================================================

``chirpy`` is a portable system for building pipelines intended for
sensing and classifying bird calls. It is intended for automated,
unattended use on sensor-class devices, doing on-device classification
of acoustic data and reporting the results wirelessly.
